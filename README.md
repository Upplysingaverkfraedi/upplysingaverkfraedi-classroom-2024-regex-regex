## Liður 1:

Í lið eitt þarf að keyra í terminali:

python3 lidur1.py --data/kt.txt --fyrirtaeki fyrir fyrirtæki

## Liður 2:

Til þess að keyra kóðann þarf að gera á skipunarlínuna:

python3 regex_email.py --file email.txt
